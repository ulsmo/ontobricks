About OntoBricks

Digital Twin Builder for Databricks — from Unity Catalog to a queryable knowledge graph in four clicks

What is OntoBricks?

OntoBricks transforms Databricks tables into a materialized knowledge graph. Design ontologies (OWL/RDFS/SHACL), map them to Unity Catalog tables via R2RML, materialize triples into a Delta-backed triple store with a Lakebase Postgres graph engine, reason over the graph, and query it through an auto-generated GraphQL API — all from a single web application running as a Databricks App.

LLM-powered automation drives the full pipeline: import metadata, generate the ontology, auto-map entities and relationships, then synchronize the Digital Twin.

The 4-Click Pipeline
  1. Import Metadata — fetch tables and columns from Unity Catalog.
  2. Generate Ontology — the LLM designs entities, relationships, and attributes.
  3. Auto-Map — the LLM produces SQL mappings for every entity and relationship.
  4. Synchronize — execute mappings and populate the triple store.
