Import R2RML Mapping

Import an R2RML (RDB to RDF Mapping Language) file to load entity and relationship mappings from an existing mapping definition. Supported formats: Turtle (.ttl) and RDF/XML (.rdf).

Importing an R2RML file will replace the current entity and relationship mappings. Make sure to save your domain first if you want to preserve the current state.
