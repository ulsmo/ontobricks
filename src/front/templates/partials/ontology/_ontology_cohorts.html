3 Link members via a shared entity (optional)

Two members are linked when they reach the same terminal entity by walking a path of object-properties (1 or more hops, e.g. Person → assignedTo → Project → governedBy → ComplianceType). Each hop can carry an optional where condition on the intermediate node (equal / in / between). Add several paths and combine them with ANY (link if any path matches) or ALL (link only if every path matches). Skip this section to treat every pair of members as linked.
